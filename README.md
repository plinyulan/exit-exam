# Exit Exam Project

This project is part of the **Exit Exam** requirement.  
It is developed as a **Full-Stack Web Application**, clearly separating Backend and Frontend, to demonstrate understanding of system design, application development, and client–server communication.

---

## Project Overview

The system is designed to demonstrate full-stack development skills with clear separation of concerns.  
The **Backend** is responsible for business logic and API services, while the **Frontend** handles user interaction and presentation through a web application.

---

## Tech Stack

### Backend
- Language: **Go (Golang)**
- Responsibilities:
  - Develop RESTful APIs
  - Handle business logic
  - Process and return data to the frontend

### Frontend
- Language: **TypeScript**
- Responsibilities:
  - User interface rendering
  - Communicate with backend APIs
  - Handle user interactions
